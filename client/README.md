# Social Media Application (Frontend)

Production-grade Social Media frontend built using Next.js 16, React 19 and modern frontend architecture.

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Redux Toolkit
- RTK Query
- Axios
- React Hook Form
- Zod
