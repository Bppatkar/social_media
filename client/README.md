# SocialSphere

Production-ready social media platform built with modern enterprise architecture.

---

# Tech Stack

## Frontend

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Redux Toolkit
- RTK Query
- Axios
- React Hook Form
- Zod
- Socket.IO Client
- shadcn/ui

## Backend

- Express.js
- MongoDB
- Redis
- JWT Authentication
- Socket.IO
- Docker
- AWS
- Swagger
- GitHub Actions

# Architecture

```
Next.js
      │
Redux Toolkit + RTK Query
      │
Axios + Socket.IO
      │
Express.js
      │
MongoDB + Redis
```
